Sobre mí

Soy Frontend Developer, apasionado por crear interfaces modernas, usables y accesibles. Me considero una persona proactiva, con capacidad de adaptación y gusto por los desafíos que implican resolver problemas reales mediante la tecnología.

A lo largo de mi formación desarrollé una base sólida en fundamentos de programación, bases de datos y metodologías de trabajo en equipo. Disfruto especialmente el proceso de transformar una idea en un producto digital claro y funcional.

Actualmente curso el último año de la Tecnicatura en Desarrollo de Software. Mi objetivo es crecer como desarrollador fullstack, aportando soluciones escalables y de calidad, y seguir fortaleciendo mis habilidades de comunicación y colaboración en equipos multidisciplinarios.

Proyectos

Sistema de gestión de inventario

Desarrollo de una aplicación web para el control de stock y productos. El sistema permite registrar artículos, organizar categorías, actualizar cantidades en tiempo real y generar reportes de inventario. Su implementación contribuyó a mejorar la organización del stock, reducir errores manuales y optimizar los tiempos de reposición.

Finance Game

Desarrollo de una aplicación web educativa con enfoque en finanzas personales. La plataforma ofrece cursos, lecciones y ejercicios interactivos que permiten a los usuarios aprender de forma dinámica y entretenida. Incorpora elementos de gamificación como progreso, logros y recompensas para motivar la continuidad en el aprendizaje.

Sistema de turnos para barbería

Diseño y desarrollo de una aplicación web para la gestión de turnos en una barbería. El sistema permite a los clientes reservar horarios de manera sencilla y a los administradores organizar la disponibilidad del personal. Incluye funcionalidades de registro de usuarios, historial de citas y notificaciones, lo que contribuye a optimizar la atención y mejorar la experiencia del cliente.

Sitios web responsivos

Desarrollo de sitios web aplicando buenas prácticas de diseño, accesibilidad y optimización para mejorar la experiencia del usuario. El resultado fueron interfaces claras, modernas y funcionales, pensadas para brindar una navegación fluida y agradable.
